# 家庭资产规划大师

一个面向 GitHub Pages 免费部署的家庭资产规划 Web 项目。首版聚焦纯前端架构，先解决可运行、可部署、可扩展的问题，再逐步补齐家庭资产录入、现金流分析和目标规划能力。

## 当前阶段

当前已落地：

- `Vite + React + TypeScript` 项目骨架
- `HashRouter` 路由结构，适配 GitHub Pages
- 首版应用导航与仪表盘页面
- 资产台账、目标规划、数据设置占位页面
- `GitHub Pages` 静态部署基础配置

后续将按项目计划继续推进：

- 统一数据模型
- 本地存储
- 资产/负债/收支录入
- 图表分析
- JSON 导入导出

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，可直接用于 GitHub Pages。

## GitHub Pages 部署说明

本项目默认使用：

- `Vite`
- `base: './'`
- `HashRouter`

这样可以避免仓库名路径和页面刷新 404 问题，更适合免费静态部署。

创建 GitHub 仓库后：

1. 推送代码到默认分支
2. 在仓库设置中启用 `GitHub Pages`
3. 使用仓库内的 Actions 工作流自动发布
