# 家庭资产规划大师 Figma 交付包

这不是可直接导入为可编辑图层的 Figma 文件，而是一份给手工重建设计使用的交付说明。

## 项目定位

- 产品名：家庭资产规划大师
- 形态：桌面工作台式家庭资产规划 Web 应用
- 风格关键词：浅色工作台、信息密度高、黑白灰为主、少量绿色/金色作为状态与收益强调
- 导航结构：左侧固定导航 + 顶部工具栏 + 中间主工作区

## 页面结构

## 视觉还原建议

- 左侧栏宽度：窄栏工作台风格，强调层级而非装饰
- 顶栏：轻量工具栏，保留面包屑、通知、诊断动作
- 卡片：圆角较大，但阴影要轻；边框可见；留白紧凑
- 标题层级：
  - 页面标题
  - 区块标题
  - 卡片标题
  - 辅助说明
  - caption / meta
- 表格：优先使用高密度、低装饰、强对齐
- 图表：以“辅助判断”为主，不要过度装饰
- 操作按钮：主按钮少而明确；更多使用行内动作或小按钮

## 设计优先级

建议在 Figma 里按这个顺序手工重画：

1. 先做全局壳层
2. 再做概览页
3. 再做资产配置页
4. 再做资产 / 实时分析 / 目标推进
5. 最后做诊断中心和数据与备份
